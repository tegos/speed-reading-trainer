# Speed Reading Trainer

Train your reading speed: text is progressively highlighted word by word at a configurable pace (words per minute).

## Features

- Adjustable speed (60–1500 wpm), play/pause, reset
- Load text from a `.txt` file (`Ctrl+O`) or paste from clipboard (`Ctrl+V`)
- Reset progress with `F5`
- Font size, text alignment, and reading-area width controls

## Development

```sh
npm install
npm run dev      # start dev server
npm run build    # production build
```

Built with React 19 + Vite.
